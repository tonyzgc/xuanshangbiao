# 商标子类选择系统

一个现代化、美观的商标子类选择和查询Web应用，支持45个商标分类的快速检索和导出。

## 功能特性

- **完整的商标分类数据**：包含45个大类、494个类似群、10,158个商品
- **固定布局设计**：类似群列表固定，商品列表固定显示
- **实时搜索**：支持按编码或名称搜索商品
- **多选功能**：可同时选择多个商标
- **分组显示**：右侧已选商标按大类自动分组
- **复制功能**：一键复制选中商标信息
- **导出功能**：支持导出为CSV文件，兼容Excel

## 数据来源

数据来源于 https://www.zcw.com.cn

## 使用说明

1. **选择商标大类**
   - 顶部显示45个商标大类，全部可见
   - 点击任意大类进入详细页面

2. **浏览类似群**
   - 左侧显示选中大类的所有类似群（4位编码）
   - 类似群列表固定显示，独立滚动
   - 点击类似群即可查看商品

3. **选择商品**
   - 右侧显示当前类似群的所有商品（6位编码）
   - 商品列表固定显示，无需滚动寻找
   - 点击商品即可选择或取消选择
   - 支持搜索功能，快速定位商品

4. **管理已选商标**
   - 最右侧显示所有已选商标
   - 按大类自动分组，便于查看
   - 单个商标可点击删除按钮移除
   - 支持一键清空所有选择

5. **导出数据**
   - 点击"导出"按钮下载CSV文件
   - 点击"复制"按钮复制到剪贴板
   - 导出包含完整信息（大类、类似群、商品编码、名称、备注）

## 数据结构

商标分类分为三级：

- **大类**（2位编码）：如 01、07、25
- **类似群**（4位编码）：如 0701、0702
- **商品**（6位编码）：如 070008、070009

## 文件说明

- `index.html` - 主页面文件
- `trademark_data.json` - 商标分类数据（已合并重复项）
- `scrape_data.py` - 数据抓取脚本
- `LICENSE` - MIT 开源许可证

## 数据更新

如需更新数据，运行：

```bash
python3 scrape_data.py
```

数据抓取脚本会自动：
1. 从知产网获取最新数据
2. 使用简短名称映射表（4字名称）
3. 自动合并重复商品编码
4. 生成 trademark_data.json 文件

## 技术栈

- 纯HTML/CSS/JavaScript，无框架依赖
- 使用现代CSS Flexbox和Grid布局
- 响应式设计，支持不同屏幕尺寸
